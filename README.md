# GNVE
The Going Nowhere Vulkan Engine is a sandbox for learning Vulkan and other game development libraries. Current dependencies are:

|Name|Version|Usage|
|-|-|-|
|[Vulkan](https://www.vulkan.org/)|1.4.304.0|Core rendering pipeline|
|[volk](https://github.com/zeux/volk)|1.4.304|Vulkan dynamic function loading|
|[VMA](https://github.com/GPUOpen-LibrariesAndSDKs/VulkanMemoryAllocator)|v3.3.0|Vulkan memory allocator|
|[GLFW](https://github.com/glfw/glfw)|3.4|Windowing & input|
|[GLM](https://github.com/g-truc/glm)|1.0.1|Graphics math|
|[fastgltf](https://github.com/spnda/fastgltf)|v0.8.0|gltf model loading|
|[thread-pool](https://github.com/bshoshany/thread-pool)|v5.0.0|Simple multithreading|
|[stb](https://github.com/nothings/stb)|802cd45|Various utilities|
|[JoltPhysics](https://github.com/jrouwe/JoltPhysics)|v5.3.0|Physics solver|
|[EnTT](https://github.com/skypjack/entt)|v3.15.0|ECS system|
|[zpp::bits](https://github.com/eyalz800/zpp_bits)|v4.5|ECS Serialization|
|[ImGui](https://github.com/ocornut/imgui)|v1.91.9b-docking|UI elements|
|[spdlog](https://github.com/gabime/spdlog)|v1.15.3|Logging|
|[Tracy](https://github.com/wolfpld/tracy/)||Profiling|
