# GNVE
The Going Nowhere Vulkan Engine is a sandbox for learning Vulkan and other game development libraries. Current dependencies are:
- [Vulkan](https://www.vulkan.org/)
- [volk](https://github.com/zeux/volk.git)
- [GLFW](https://github.com/glfw/glfw.git)
- [GLM](https://github.com/g-truc/glm.git)
- [ImGui](https://github.com/ocornut/imgui.git)
- [EnTT](https://github.com/skypjack/entt.git)
- [JoltPhysics](https://github.com/spnda/fastgltf.git)
- [fastgltf](https://github.com/zeux/volk.git)
